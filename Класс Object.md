Метод equal сравнивает две ссылки на объекты, если эти объекты одинаковые, вернёт True или False.

Метод hashCode выдаёт числовое значение, соответствующее данному объекту, главное чтобы у разных объектов это число отличалось, а у одинаковых совпадало.

Метод toString переводит любой объект в строку, работает странно, чаще всего для использования придётся переписать.
